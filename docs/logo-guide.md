# Logo WallOs

## Archivos

- Icono: apps/web/public/brand/wallos-mark.svg
- Wordmark: apps/web/public/brand/wallos-logo.svg
- Favicon: apps/web/public/favicon.svg

## Concepto

El isotipo combina:

- Una forma redondeada tipo app de macOS.
- Un monograma W con trazos suaves.
- Una linea de horizonte que sugiere paisaje y wallpaper.

## Uso recomendado

- Usar wallos-mark.svg para avatares, iconos y espacios pequenos.
- Usar wallos-logo.svg en navbar, hero y material de marketing.
- Mantener zona de seguridad equivalente al 20% del ancho del logo.
- No alterar colores base del gradiente en piezas principales.
